# Part Sourcing Compare

여러 전자부품 소싱 사이트를 한 번에 검색하고, 가격·재고·MOQ·리드타임을 하나의 표에서 비교하는 웹 도구입니다.
별도 설치 없이 `index.html` 파일 하나로 PC 브라우저에서 바로 동작합니다.

## 주요 기능

- **부품번호 입력 한 번**으로 여러 사이트의 검색 결과 페이지를 한꺼번에 열기
- **자동 / 수동 사이트 통합 비교표** — 단가(USD)·재고·MOQ·리드타임·메모를 한 표에서 관리
- **예상금액 자동 계산** (단가 × MOQ)
- **최저 단가 자동 표시** (노란색 + BEST)
- **입력값 자동 저장** (브라우저 localStorage)
- **CSV 내보내기** 및 **PDF 인쇄/저장**으로 공유
- **사이트 직접 추가·삭제** (거래처 자유 등록)

## 사용 방법

1. `index.html` 을 브라우저에서 엽니다.
2. 부품번호를 입력하고 **검색표 만들기**를 누릅니다.
3. **자동사이트 전체 열기**로 결과 페이지들을 확인한 뒤, 가격·재고를 표 칸에 입력합니다.
4. 필요 시 **CSV 내보내기** 또는 **인쇄/PDF**로 결과를 저장·공유합니다.

## 참고

- 본 도구는 각 사이트의 공개(로그인 불필요) 검색 페이지로 연결하는 런처입니다.
- 가격·재고 수치는 직접 확인 후 입력하는 방식입니다. (브라우저 보안(CORS)상 외부 사이트 데이터의 자동 수집은 공식 API 키가 있어야 가능합니다.)

## GitHub Pages 공개 (선택)

저장소 Settings → Pages → Branch를 `main` / `/(root)` 으로 설정하면
`https://<사용자명>.github.io/<저장소이름>/` 주소로 바로 접속할 수 있습니다.
